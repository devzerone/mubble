# mubble

마크다운 기반 터미널 애플리케이션 - 파일 경로를 자동으로 감지하고 바로 열 수 있는 스마트 터미널

## 🎯 프로젝트 목표

- 마크다운 형식의 입력을 지원하는 터미널
- 파일 경로(상대경로)를 자동으로 감지하고 하이퍼링크로 변환
- 클릭 한 번으로 파일 바로 열기
- 개발자 친화적인 터미널 경험

## ✨ 핵심 기능

1. **마크다운 입력**: 실시간 마크다운 렌더링과 문법 하이라이팅
2. **경로 자동 감지**: `./file.md`, `../src/main.go` 등의 경로를 자동으로 감지
3. **빠른 파일 열기**: 클릭 한 번으로 파일 열기 (라인 번호 지원)
4. **터미널 명령**: 일반 터미널처럼 명령 실행
5. **테마 지원**: 밝은/어두운 테마, 커스텀 색상

## 🚀 빠른 시작

### 설치

```bash
# Go 1.21+ 필요
go install github.com/devzerone/mubble@latest
```

### 실행

```bash
mubble
```

### 사용 예시

```bash
# 일반 터미널 모드 (기본)
$ ls -la
$ npm run dev

# 마크다운 모드 진입 (Ctrl+M)
# 분할 화면으로 전환: 입력창 + 미리보기
> # 프로젝트 문서
> 이것은 **볼드** 텍스트입니다
> - 리스트 항목 1
> - 리스트 항목 2

# 마크다운 모드 종료 (Ctrl+M)
# 다시 일반 터미널 모드로 복귀

# 파일 경로 자동 감지 (마크다운 모드에서)
> ./docs/README.md      # 클릭하여 열기
> ../src/main.go:25     # 25번째 줄로 이동하여 열기
```
