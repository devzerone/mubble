## 버전 규칙

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다:

- **MAJOR**: 기존 기능의 호환되지 않는 변경
- **MINOR**: 새로운 기능 추가 (기존 기능과 호환)
- **PATCH**: 버그 수정

## 변경 유형

- **Added**: 새로운 기능 추가
- **Changed**: 기존 기능의 변경
- **Deprecated**: 향후 제거될 기능
- **Removed**: 제거된 기능
- **Fixed**: 버그 수정
- **Security**: 보안 관련 수정
